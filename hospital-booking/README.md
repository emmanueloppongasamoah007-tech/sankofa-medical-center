# 🏥 Sankofa Medical Center
### Hospital Appointment Booking System

A web-based hospital appointment booking application that allows patients to book appointments online, browse departments, and manage their profiles. Administrators have a dedicated dashboard to manage all bookings.

Built as a Capstone Project for **BIT 293 - Web-Based Concepts & Technology I**
**Kwame Nkrumah University of Science and Technology (KNUST), March 2026**

---

## 🌐 Live Pages
- Home
- About
- Departments & Services
- Book Appointment
- Contact
- Login / Sign Up
- Admin Dashboard

---

## ✨ Features
- Online appointment booking form with date and time selection
- Patient login and registration system
- Administrator dashboard to view and cancel appointments
- Hospital departments and services listing
- Contact form
- Fully responsive design (mobile, tablet, desktop)
- Client-side form validation
- localStorage for data persistence (no backend required)

---

## 🛠️ Technologies Used
- **HTML** — Page structure and semantics
- **CSS** — Styling, layout (Flexbox & Grid), responsive design
- **JavaScript** — Form validation, DOM manipulation, localStorage

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/emmanueloppongasamoah007-tech/sankofa-medical-center.git
   ```
2. Open the project folder in VS Code
3. Run with Live Server:
   ```bash
   live-server
   ```
4. Open `http://127.0.0.1:8080/hospital-booking/` in your browser
